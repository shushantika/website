HTML Terms

HTML
Hyper Text Markup Language
HTML element
Everything from the start tag to the end tag. Like <p> this is a paragraph element </p>.
HTML tag
Keywords surrounded by angle brackets like <html>.
angle bracket
< and > are used to enclose HTML tags.
HTML attribute
Provide additional information about HTML elements which come in name/value pairs. Like name = value.
HTML comment
Information that is not displayed, but has helpful information for coder. Use < !--...-- >.
HTML object element
Elements that insert objets rather than enclose text. They do not have a closing tag
HTML special character
A character that cannot be printed out by itself because of its reserved functions
